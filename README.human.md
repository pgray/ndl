# ndl - needle

needle is a TUI for threads

if you are working and want to be aware of notifications but decrease distractions somewhat, maybe this will help

# install

- for linux: `cargo install ndl`
- for mac: maybe that works
- for windows: good luck

# usage

- h, j, k, l navigation (left, down, up, right respectively)
- q to quit
- esc to exit an edit
- more to come
